# 被動元件產業研究筆記
整理日期：2026-05-14

## 產業現況

### 全球市場規模
- MLCC（積層陶瓷電容）2025年市場規模：約163.9億美元
- MLCC 2026年預估：176.1億美元
- 預計2036年超過353.1億美元，CAGR 7.2%（2026-2036）
- 整合式被動元件2025年15.9億美元，2035年達33.7億美元

### 台灣產業數據
- 台灣被動元件2026年產值預估：約2,559億元，年增6.4%
- AI與車用為雙引擎

### 驅動因素
1. AI伺服器：GB200等高階AI伺服器MLCC用量是傳統伺服器的2-3倍
2. 電動車：高階MLCC需求升級
3. 原物料價格：銀價大漲，推動漲價潮
4. 供應鏈紀律：日、韓大廠擴產保守

### 價格趨勢（2024-2026漲價潮）
- 2024年7月：積層式電感、磁珠漲價15-20%（村田、TDK、國巨、華新科）
- 2025年6月：鉭質電容漲價合計達30%（國巨、AVX、Vishay）
- 2026年1月：Panasonic鉭質電容漲15-30%；中國晶片電阻廠漲8-20%
- 2026年2月：MLCC醞釀漲價15-20%（國巨、華新科、禾伸堂）
- 2026年3-4月：村田、三星電機高階MLCC需求強勁

### AI應用關鍵數據
- 村田AI相關MLCC訂單達產能2倍，BB值1.07（15季新高）
- 2026年八大CSP資本支出8,100億美元，持續推升需求
- 高端AI伺服器MLCC用量20-30萬顆，是普通伺服器10倍、手機30倍

## 風險因素
1. 地緣政治與關稅政策不確定性
2. 原物料（銀、銅、鋁）價格持續上漲
3. 消費性電子需求仍弱
4. 景氣循環風險

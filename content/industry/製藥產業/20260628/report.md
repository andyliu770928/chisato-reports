# 全球製藥產業研究報告

**日期：2026年6月28日**
**研究主題：全球製藥市場（製藥產業）**

---

## 摘要

全球製藥市場是全球最大、影響力最深的產業之一，涵蓋藥物發現、開發、生產與上市銷售。2022年市場規模約**1.48兆美元**，已開發市場（美國、歐洲、中國）占據主導地位。Biologics（小分子藥物）仍為主流，但生物製劑、快速成長的學名藥市場，以及AI驅動的藥物發現技術，正在重塑整個產業的競爭格局。

---

## 1. 市場規模與成長

### 1.1 全球市場數據

| 指標 | 數據 |
|------|------|
| 市場規模（2022） | 約 **1.48 兆美元**（Wikipedia） |
| 市場規模（2020→2021） | 1,228 億 → 1,250 億美元（GlobeNewswire） |
| 複合年成長率（CAGR） | 約 **5.5–6%**（各機構預測不一致，區間5.5%–12%） |
| 市場估值（2021） | 約 **7.03 兆美元**（Torreya），公開交易的製藥公司約6.1兆 |
| 2030 預測 | 根據不同機構，落在 1.6 兆至 2.0 兆美元區間 |

> 📌 製藥市場有多個口徑：狹義（專利藥）與廣義（含學名藥、生物製劑）差異大，不同機構預測差異亦大，引用時需標明來源。

### 1.2 區域市場

- **北美（特別是美國）**：全球最大單一藥品市場，擁有最高藥價與最多創新藥研發預算
- **歐洲**：德國、瑞士、法國、英國為主要市場，監管框架（EMA）與美國（FDA）並列全球两大藥政主管機關
- **亞太區**：成長最快的市場，主要驅動來自：
  - 中國：製藥業估值佔全球從 2003 年的 1% 上升至 2021 年的 **12%**，已超越瑞士
  - 印度：學名藥（Generic Drugs）製造重鎮
  - 日本：老齡化社會支撐持續用藥需求

### 1.3 製藥次產業趨勢

| 次產業 | 變化趨勢 |
|--------|----------|
| **小分子藥物（Small Molecules）** | 估值佔比從 2003 年的 84.6% 降至 58.2%（2021） |
| **生物製劑（Biologics）** | 估值佔比從 14.5% 升至 30.5%（2021），持續快速成長 |
| **學名藥（Generic Drugs）** | 專利懸崖效應下，比重穩定上升，價格壓力加劇 |
| **動物保健（Animal Health）** | 穩定成長，寵物藥物需求上升 |
| **中國製藥** | 快速崛起，估值佔比從 1%→12%，為值得關注的新興力量 |

### 2.1 價值鏈

```

    ↓
臨床前試驗（in vitro / in vivo）
    ↓
臨床試驗（Phase I → II → III）
```

## 3. 核心技術與研發動態

### 3.1 藥物發現與開發成本

| 階段 | 說明 |
|------|------|
| 研發費用 | 業界 R&D 投入於 2009 年達 **653 億美元**歷史高點 |
| 每顆新藥開發成本 | 估算區間 **40 億至 110 億美元**（含失敗候選藥物的攤提） |
| 開發時程 | 通常 **10–15 年**（從發現到核准） |
| FDA 新藥核准數 | 2010 年：18 個新分子實體（NME）+ 3 個生物製劑 = 21 個 |

> 📌 藥物開發失敗率高（約 90% 的候選藥物在臨床試驗中失敗），是成本高昂的主因。

### 3.2 AI 與藥物發現（2020年代最大趨勢）

AI 正在根本性改變藥物發現的速度與成本：

- **AlphaFold（DeepMind）**：預測蛋白質三維結構，解決數十年來的结构生物学难题
- **Exscientia（英國）**：AI 設計分子，進入臨床試驗候選藥物
- **Insitro（美國）**：結合 ML 與基因體學設計藥物
- **Schrödinger（美國）**：計算化學 + AI 加速候選藥物篩選
- **輝瑞、GSK、Sanofi**：積极與AI新創合作或併購

### 3.3 細胞與基因療法

- **CAR-T 療法**：Kymriah（Novartis）、Yescarta（Gilead）已獲核准，治療血癌
- **基因編輯（CRISPR）**：Vertex + CRISPR Therapeutics 的 Casgevy（2023年核准）為首款 CRISPR 藥物
- **mRNA 技術**：COVID-19 疫苗驗證 mRNA 平台可行性，擴展至癌症免疫療法

### 3.4 製藥製造趨勢

- **原料藥（API）外包**：印度與中國已是全球最大 API 生產基地，成本壓力持續
- **連續式製造（Continuous Manufacturing）**：FDA 積極推動，取代傳統批次製造
- **生物製劑產能擴張**：各大藥廠積極興建生物製劑廠（Biomanufacturing）

## 4. 監管環境

### 4.1 主要藥政法規

| 機構 | 地區 | 角色 |
|------|------|------|
| **FDA（美國食品藥物管理局）** | 美國 | 全球最嚴格、最具影響力的藥政法規機構 |
| **EMA（歐洲藥品管理局）** | 歐盟 | 歐洲單一藥證審批 |
| **NMPA（國家藥品監督管理局）** | 中國 | 中國市場准入，曾因疫苗醜聞（長生生物2018）加強監管 |
| **PMDA（醫藥品醫療機器總合機構）** | 日本 | 日本藥證審批 |
| **TFDA（衛生福利部食品藥物管理署）** | 台灣 | 台灣藥證 |

### 4.2 重要法規里程碑

- **1938 年 FD&C Act（美國）**：要求藥物安全性，源於磺胺劑悲劇
- **1962 年 Kefauver-Harris Amendment（美國）**：要求藥物有效性證明，源於沙利竇邁悲劇
- **2016 年 21st Century Cures Act（美國）**：加速藥證審批，支持罕見疾病藥物
- **2019 年 DRSS（中國）**：醫保藥品目錄談判機制，顯著壓低藥價

### 4.3 藥價壓力

- **美國**：通膨抑制法案（IRA, 2022）赋予 Medicare 談判藥價的權利，衝擊製藥業
- **歐洲**：各國價格談判機制，藥價普遍低於美國
- **中國**：醫保談判大幅降價（PD-1抑制劑從每年百萬降至數萬人民幣）
- **學名藥競爭**：專利到期後價格通常下跌 80–90%

## 5. 重大課題與挑戰

### 5.1 專利懸崖（Patent Cliff）

| 藥物 | 藥廠 | 專利到期年份 | 影響 |
|------|------|-------------|------|
| Humira | AbbVie | 2023 | 史上最大學名藥衝擊 |
| Keytruda | Merck | 2028 | 預計數百億美元銷售面臨威脅 |
| Eliquis | BMS/Pfizer | 2027 | 抗凝血藥市場 |
| Revlimid | BMS | 2025 | 多發性骨髓瘤藥物 |

### 5.2 主要挑戰

1. **研發生產力危機**：尽管研发投入創新高，新藥核准數並未相應增加
2. **藥價談判壓力**：政府醫保機構越來越積極干預藥價
3. **生物相似藥（Bio-similars）競爭**：原廠生物製劑專利到期後，生物相似藥搶市
4. **供應鏈安全**：COVID-19 暴露原料藥過度依賴中國/印度的風險，美歐日積極推動製藥供應鏈回流
5. **AI 藥物發現的不確定性**：AI 候選藥物臨床試驗失敗案例陸續出現
6. **罕見疾病藥物開發**：誘人的監管優惠（優先審批、費用補助）與極小的市場之間的平衡
